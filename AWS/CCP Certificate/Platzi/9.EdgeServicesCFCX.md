# Edge Services in AWS and how to improve user experience

This kinds of services improve latency and eficiency in the access to aws resources.

## AWS Edge Services
### 1. Amazon CloudFront
***Definition: Amazon CloudFront is a content delivery network (CDN) service that securely delivers data, videos, applications, and APIs to customers globally with low latency and high transfer speeds.***

Use cases:
- Delivering static and dynamic web content.
- Streaming video and audio.
- Security certificates with AWS Certificate Manager (ACM).

***Content closer to the user***

### 2. AWS Global Accelerator
***Definition: AWS Global Accelerator is a networking service that improves the availability and performance of your applications with local or global users.***
Use cases:
- Improving application availability.
- Reducing latency for global users.
- Providing static IP addresses for applications.

Improves the route to the application using the AWS global network.
